# MBTI Vercel App

Aplikasi laporan MBTI berbasis React/Vite.

## Jalankan lokal
```bash
npm install
npm run dev
```

## Deploy ke Vercel lewat GitHub
1. Upload seluruh folder ini ke repository GitHub.
2. Buka Vercel > Add New Project > pilih repository.
3. Framework akan terdeteksi sebagai Vite.
4. Build Command: `npm run build`
5. Output Directory: `dist`
6. Klik Deploy.

## Cara pakai
Isi data peserta dan skor pasangan I/E, S/N, T/F, J/P. Setiap pasangan mengikuti logika Excel: total 15.
